Erios is a game with many features and systems that all work together. It's ultimately one big sandbox, where a subset of these features and systems are used to create the gameplay experience. Almost every system in Erios can be extended.

The most common example would be a level. When you play Erios, you are always within a level. The entire region of Weldanar? It's one big level with a massive terrain fulled with thousands of objects. You can also make levels yourself, and link these to the world. Ever wanted to add an extra passage to the Cave of Allunissa, create a dungeon or just add some extra housing in Hergen? It's all possible thanks to the in-game **Mod Editor**

## Mod Editor

Whereas some games need external tools to change the game, Erios has a built-in Mod Editor!
