Below is the complete list of reserved words recognized by Lunaris. Reserved words cannot be used as the name of a variable, class, or function.

Links point to the most relevant section in the manual. Some entries are aliases or syntax helpers, so they link to the feature they belong to.

| Keyword | Documentation | Kind | Description |
| ------- | ------------- | ---- | ----------- |
| `attempt` | [Attempt](../19. Error Handling/#attempt) | Keyword | Returns `nil` when an expression throws |
| `break` | [Break](../5. loops/#keyword-break) | Keyword | Stops the current loop or `switch` case flow |
| `case` | [Switch](../4. conditions/#switch) | Keyword | Declares a branch inside a `switch` |
| `catch` | [Try & Catch](../19. Error Handling/#try--catch) | Keyword | Handles an exception thrown in a `try` block |
| `class` | [Classes](../9. classes) | Keyword | Declares a class |
| `const` | [Constants](../8. constants) | Keyword | Marks a value as immutable |
| `continue` | [Continue](../5. loops/#keyword-continue) | Keyword | Skips to the next loop iteration |
| `else` | [Else](../4. conditions/#else) | Keyword | Runs when previous conditions did not match |
| `elseif` | [Elseif](../4. conditions/#elseif) | Keyword | Adds another conditional branch after `if` |
| `enum` | [Enums](../10. enums) | Keyword | Declares an enum |
| `exception` | [Exception](../19. Error Handling/#exception) | Keyword | Declares a custom exception type |
| `extension` | [Extensions](../12. extensions) | Keyword | Extends an existing type with methods |
| `false` | [Bool](../2. types/#bool) | Literal | Boolean literal for false |
| `finally` | [Finally](../19. Error Handling/#finally) | Keyword | Runs cleanup code after `try` / `catch` |
| `for` | [For](../5. loops/#for) | Keyword | Repeats code over a range or collection |
| `function` | [Function](../6. functions/#function) | Keyword | Declares a function |
| `goto` | [Goto](../11. goto) | Keyword | Jumps to a label |
| `if` | [If](../4. conditions/#if) | Keyword | Starts a conditional statement |
| `in` | [For](../5. loops/#for) | Keyword | Iteration keyword used in `for (... in ...)` |
| `local` | [Variables](../1. variables) | Keyword | Declares a local variable |
| `mixin` | [Classes](../9. classes) | Keyword | Declares a mixin for class composition |
| `new` | [Constructor](../9. classes/#constructor) | Keyword | Creates a new instance of a class |
| `nil` | [Nil](../2. types/#nil) | Literal | Literal representing no value |
| `private` | [Private](../9. classes/#private) | Keyword | Restricts a member to the declaring class |
| `protected` | [Protected](../9. classes/#protected) | Keyword | Restricts a member to the class and derived classes |
| `public` | [Public](../9. classes/#public) | Keyword | Exposes a member publicly |
| `repeat` | [Repeat](../5. loops/#repeat) | Keyword | Starts a repeat-until loop |
| `return` | [Return value](../6. functions/#return-value) | Keyword | Returns a value from a function |
| `sealed` | [Sealed](../9. classes/#sealed) | Keyword | Prevents inheritance or overriding |
| `static` | [Static](../9. classes/#static) | Keyword | Marks a class member or class as static |
| `switch` | [Switch](../4. conditions/#switch) | Keyword | Compares one value against multiple branches |
| `throw` | [Throw](../19. Error Handling/#throw) | Keyword | Throws an exception |
| `true` | [Bool](../2. types/#bool) | Literal | Boolean literal for true |
| `try` | [Try & Catch](../19. Error Handling/#try--catch) | Keyword | Starts protected code that may throw |
| `until` | [Repeat](../5. loops/#repeat) | Keyword | Ends a `repeat` loop when a condition becomes true |
| `while` | [While](../5. loops/#while) | Keyword | Repeats code while a condition is true |
